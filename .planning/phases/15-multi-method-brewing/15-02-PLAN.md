  `bloom_weight: Optional[float] = Form(None)` and `brew_volume: Optional[float] = Form(None)`
- Build measurement_data based on method:
  ```python
  if method == "pour-over":
      measurement_data = {

          "brew_volume": brew_volume or 250.0,

      measurement_data = {
          "grind_setting": grind_setting,
          "temperature": temperature,
          "preinfusion_pct": preinfusion_pct,
          "dose_in": dose_in,
          "target_yield": target_yield,
          "saturation": saturation,
          "taste": taste,
      }
  ```

**`manual_brew()`** (GET):
